Atlas RP v0.8.4 — Import / Export

- Réservé aux administrateurs en mode Administrateur.
- L’export génère AtlasRP_Backup_YYYY-MM-DD.json.
- L’import ajoute les données à l’Atlas existant et remappe les identifiants.
- Les fichiers médias sont recopiés dans le bucket atlas-media quand leur URL publique est accessible.
- Un import dans le même Atlas peut créer des doublons de marqueurs et de zones.
